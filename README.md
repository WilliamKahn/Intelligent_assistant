## 项目概述
基于 Hamibot 开发的自动化脚本系统，通过智能算法实现：
- 自动运行 **30+ 移动应用**（涵盖主流阅读和视频应用）
- 动态计算应用收益价值（单位时间收益/任务稳定性/时间成本）
- 智能调度任务执行顺序（收益优先 + 账号封禁状况）
- 每日收益推送

## 安装说明
